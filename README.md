# Customer Shopping Behavior Analysis

## Project Overview

This project focuses on analyzing customer shopping behavior using Python, SQL, and Power BI. The objective is to transform raw data into meaningful insights through data cleaning, exploratory data analysis (EDA), SQL querying, and interactive dashboard creation.

## Objectives

* Understand customer purchasing patterns and demographics.
* Clean and prepare raw data for analysis.
* Perform exploratory data analysis to identify trends and relationships.
* Use SQL to answer business-related questions.
* Build an interactive Power BI dashboard for visualization.
* Generate a comprehensive analytical report.

## Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **SQL**
* **Power BI**
* **Jupyter Notebook**

## Project Workflow

### 1. Data Loading

* Imported the dataset using Python Pandas.
* Inspected data structure, columns, and data types.

### 2. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent data formats.
* Validated data quality for accurate analysis.

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer demographics.
* Examined purchasing behavior and spending patterns.
* Identified key trends and correlations.
* Created visualizations to support findings.

### 4. SQL Analysis

Performed SQL queries to answer business questions such as:

* Customer distribution by age group.
* Spending patterns across categories.
* Most popular products and categories.
* Customer segmentation insights.

### 5. Power BI Dashboard

Created an interactive dashboard featuring:

* Sales and purchase trends.
* Customer demographics analysis.
* Category-wise performance.
* Key Performance Indicators (KPIs).
* Dynamic filters and slicers.

### 6. Reporting

Prepared a business-focused report summarizing:

* Major findings.
* Customer behavior insights.
* Data-driven recommendations.
* Dashboard highlights.

## Key Insights

* Identified major customer segments based on purchasing behavior.
* Discovered top-performing product categories.
* Analyzed spending patterns across different age groups.
* Highlighted opportunities for improving customer engagement and sales performance.
